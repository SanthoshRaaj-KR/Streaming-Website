# Streaming Website

This is a **Streaming Website** project that includes both **backend** and **frontend** components. It is designed to allow users to stream videos with an intuitive interface.

## Project Overview

This project is a **work in progress** and aims to provide users with the ability to stream content, search for videos, and enjoy an interactive user experience. The backend is built using **Express.js** (Node.js), and the frontend is developed using **EJS** templates, **JavaScript**, and **CSS** for styling.

### Key Features:
- User-friendly video streaming interface.
- Video search functionality.
- Responsive design for different screen sizes.
- Simple backend with Express.js to handle user requests and serve content.

## Technologies Used

- **Backend**: 
  - [Node.js](https://nodejs.org/) 
  - [Express.js](https://expressjs.com/)
  
- **Frontend**:
  - [EJS](https://www.npmjs.com/package/ejs) (Embedded JavaScript templating engine)
  - [JavaScript](https://www.javascript.com/)
  - [CSS](https://www.w3.org/Style/CSS/) (for styling)
